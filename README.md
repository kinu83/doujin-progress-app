# 同人原稿進捗管理アプリ 『修羅場メーター』　

### 「原稿の"間に合う気がする"を、現実にする」

修羅場メーターは、同人原稿における**締切までの作業時間を「修羅場レベル」として可視化**する、原稿進捗管理Webアプリです。

ページ数や工程ごとの作業時間、締切までの日数をもとに、残り作業時間や今日必要な作業時間を自動で計算します。

**締切までの不安を減らし、原稿を進めるのが楽しくなるようなアプリ**を目指しています。

## 背景
自分が同人活動を行う中で、原稿制作だけでなく、イベント申し込み、印刷所決め、入稿準備、当日の準備、在庫管理など、さまざまなことを1人で行う必要がありました。

特に複数冊を制作する場合、予定を立てる難しさを感じることが多く、スケジュールを細かく管理する余裕がなくなることもありました。また、残りの作業量や必要な時間が見えにくく、時間の捻出にも難しさを感じていました。

そこで、簡単に同人誌制作のスケジュールを立てられるアプリ、そして原稿へのモチベーションを保てるようなアプリが欲しいと考え、実際に制作に取り組みました。

## このアプリの特徴

### 「同人誌制作」に特化している

原稿制作における工程ごとに、ユーザーが自由に作業時間を設定できます。
ネーム、下書き、ペン入れ、仕上げなど、自分の制作スタイルに合わせた進捗管理ができます。

また、進捗はページ単位で管理できるため、同人誌制作の流れに合わせて原稿の進み具合を確認できます。

### 作業時間を自動計算

工程ごとの作業時間、締切までの日数、ページ数をもとに、今日やらなければいけない作業時間を自動で算出します。

その作業時間をもとに修羅場レベルを表示し、今の進捗がどれくらい危ないのかを分かりやすく確認できます。
修羅場レベルの判定基準も変更できるため、自分の作業ペースに合わせた管理ができます。

### 誰でも始めやすい設計

新規作成から実際の進捗管理まで、できるだけ簡単に操作できるようにしました。

必要な情報を入力するだけで、すぐに原稿の進捗状況や今日必要な作業時間を確認できます。


## 使用技術

### フロントエンド
- Nuxt 4
- Vue 3
- TypeScript
- Tailwind CSS 4

### バックエンド・インフラ
- Firebase Authentication
- Cloud Firestore
- Firebase Hosting

## 実装した機能

### 認証
- 匿名・Googleログイン
- ログイン状態の保持
- ユーザーごとのデータ管理

### プロジェクト管理
- プロジェクトの作成・編集・削除
- 締切日・イベント日・ページ数の設定
- 本の仕様などのメモ欄

### 進捗入力・進捗計算機能
- ページごとの進捗入力
- 1日あたりの必要作業時間の算出
- 修羅場レベルの判定

### データ保存機能
- Firestoreへのプロジェクト保存
- Firestoreへの進捗データ保存
- ユーザーごとのデータ分離

## 進捗計算ロジック

- 残り日数 = 締切日 - 今日の日付
- 残り作業時間 = 未完了工程の想定作業時間の合計
- 1日あたりの必要作業時間 = 残り作業時間 ÷ 残り日数
- 修羅場レベル = 1日あたりの必要作業時間をもとに判定
