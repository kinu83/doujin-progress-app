# 修羅場メーター

## 概要

**修羅場メーター**は、同人誌・漫画原稿の進捗管理を支援するWebアプリです。

ページごとの進捗や工程別の作業時間をもとに、残り作業時間・今日必要な作業時間・締切までの危険度を自動で計算します。  
同人創作者が締切までの状況を直感的に把握し、無理のない制作計画を立てられることを目指して制作しました。

## 制作背景

同人誌制作では、ネーム・下書き・線画・仕上げなど複数の工程があり、ページごとに進捗が異なります。  
そのため、単純なToDo管理では「今どれくらい進んでいるのか」「今日どれだけ作業すれば間に合うのか」が分かりにくいと感じました。

そこで、原稿制作に特化した進捗管理アプリとして本アプリを開発しました。

## 主な機能

- 原稿プロジェクトの作成・編集・削除
- イベント日・入稿締切日・作業開始日の管理
- ページごとの進捗ステータス管理
- 工程ごとの作業時間をもとにした進捗率の計算
- 残り作業時間・今日必要な作業時間の自動算出
- 締切までの危険度を示す「修羅場レベル」の表示
- カレンダーでの日別予定・実績作業時間の管理
- 本文仕様、印刷会社、部数、予算などのメモ管理
- 匿名ログイン・Googleアカウント連携
- Firestoreへのデータ保存

## 工夫した点

### 原稿制作に合わせた進捗管理

一般的なタスク管理ではなく、同人誌制作の流れに合わせて、ページ単位・工程単位で進捗を管理できるようにしました。

### 締切までの状況を可視化

残り作業時間と締切までの日数から、今日必要な作業時間を計算し、修羅場レベルとして表示しました。  
これにより、ユーザーが現在の危険度を直感的に把握できるようにしています。

### すぐに使える認証設計

初回利用のハードルを下げるため、匿名ログインで利用できるようにしました。  
あとからGoogleアカウントに連携することで、データを引き継げる設計にしています。

## 画面構成

- `/`  
  プロジェクト一覧

- `/projects/new`  
  新規プロジェクト作成

- `/projects/[id]`  
  プロジェクト詳細・ページ進捗管理

- `/calendar`  
  日別の予定・実績作業時間管理

- `/settings`  
  作業工程や修羅場レベルの設定
